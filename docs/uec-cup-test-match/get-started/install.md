## 手順１

NOTE: Windows に付属している telnet をアンインストールする方法を説明します。なぜなら、 Windows に付属している telnet を使っても、おかしな挙動をして難しいから。  

Windows のスタート・ボタンの横の検索ボックスに `control` と打鍵。  
［コントロール パネル］をクリック。  
［プログラム　＞　Windows の機能の有効化または無効化］をクリック。  

- ［Telnet Client］チェックボックスにチェックが付いていたら：　チェックを外す。  
    - ［OK］ボタンをクリック。  
    - アンインストールが始まるのでしばらく待つ。  
- そうでなければ：
    - 何もありません。

［閉じる］ボタンをクリック。  

これで Windows に付属している telnet のアンインストールが終わりました。
